LLM and Agentic Evaluation Dashboard

This is a browser-only version of the project so you can host it on GitHub Pages. It explains the core ideas, lets users download sample datasets, upload edited JSON or CSV files, and runs simple evaluation directly in the browser.

No API keys are required here. The page evaluates answers already present in the uploaded dataset, which makes it easy to learn the concepts before wiring up live models.

How Answers Are Represented In This Demo

LLM

User question -> model -> answer

The answer is already stored in the dataset as actual_answer.

RAG

User question -> retriever -> context -> model -> answer

The retrieved chunks are stored in retrieved_contexts.

Agentic

Task -> planner -> tools -> model -> final result

The workflow trace is stored in steps.
